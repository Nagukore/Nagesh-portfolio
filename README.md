## 🛠️ Technology Stack
- **Framework:** React 18
- **Language:** TypeScript
- **Tooling:** Vite
- **Styling:** Elegant custom Vanilla CSS with glassmorphism interactions and fully custom dynamic CSS animations.
- **Icons:** Lucide-React

## ✨ Key Features
- **NagiBot Interface:** Features a highly interactive, built-in smart chatbot complete with developer jokes, savage roasts, and easter eggs to entertain visitors.
- **Sleek Aesthetic:** Modern, premium dark-mode presentation with an immersive ambient floating dust and pulsing colored orb background animation.
- **Dynamic Modals:** Project cards seamlessly fetch and render live, cleanly-formatted Markdown descriptions upon interaction.
- **Interactive Forms:** Implements animated floating-label Material-style inputs for an elite contact outreach experience.
- **Dedicated Experience Section:** Showcases professional milestones including my ThoughtsCrest internship and independent full-stack architecture roles.
